Projekte

2025 – 2026
FEATUREDSecureVault Cloud — Serverlose Datei-Speicherplattform auf AWS
securevault-cloud.vercel.app · github.com/universw/securevault-cloud
  • Konzeption und Bereitstellung einer Full-Stack-Serverless-Anwendung auf AWS (us-east-1): Frontend mit React 18 + Vite auf Vercel mit GitHub-Auto-Deploy, Backend in Python 3.12 (AWS Lambda) hinter API Gateway.
  • Implementierung einer nutzerbezogenen Dateiisolation in S3 mittels Cognito-User-IDs als IAM-bezogene Pfad-Präfixe (uploads/{userId}/); Cross-User-Zugriff wird bereits auf Speicherebene unterbunden.
  • Entwurf eines Presigned-URL-Upload-/Download-Flows: Dateien werden direkt vom Browser an S3 gestreamt, ohne Lambda zu durchlaufen — geringere Compute-Kosten, kleinere Angriffsfläche, 5-minütiger Linkablauf.
  • Aufbau von vier Lambda-Funktionen (upload-url, list, download-url, delete) mit JWT-Validierung gegen Cognito; CORS-restriktiver S3-Bucket sowie End-to-End-CI/CD via GitHub → Vercel.
